# Hand-movement-Detection-
Real-time Hand Detection using Python, OpenCV, and MediaPipe. Detects and tracks hand landmarks live through a webcam.
## Output 
![output](Handdetectionoutput.jpeg)

## Features
- Detects multiple hands simultaneously.
- Tracks 21 hand landmarks in real-time.
- Visualizes landmarks on webcam feed.
- Lightweight and fast for live applications.

## Requirements
- Python 3.10+  
- OpenCV (`opencv-python`)  
- MediaPipe (`mediapipe`)
